# 📖 Overview

VYTAL is an AI-powered athlete health monitoring system developed as a graduation project at Umm Al-Qura University.

The platform combines Business Intelligence and Artificial Intelligence to transform physiological measurements into meaningful health insights. Using Microsoft Power BI dashboards, Python integration, and Google Gemini, the system automatically analyzes athlete data, classifies health risks, and generates AI-assisted health reports.

The platform is designed to support coaches, sports analysts, and medical teams by providing interactive dashboards and evidence-based athlete health monitoring.

# 🤖 AI Integration

Python is integrated directly inside Microsoft Power BI to communicate with Google Gemini API.

The AI module performs the following tasks:

- Reads athlete physiological measurements
- Applies evidence-based thresholds
- Classifies health status
- Generates concise health interpretations
- Produces AI-powered recommendations
